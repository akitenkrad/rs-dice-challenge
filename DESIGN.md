# 概要

複数のサイコロを振って，同じ目が出るまで試行を繰り返すコマンドラインツール．
指定された秒数の間隔で定期的に施行を繰り返す．
ゾロ目が出たら停止する．

# Input

- サイコロの数
- 試行を繰り返す間隔 (秒)

# Output

一回の試行で以下のテキストを出力する．

```
⚀ ⚁ ⚂ ⚃ ⚄ ⚅ ... N ★ (ゾロ目が出た場合に星を表示する)
number of trials: XXXXXXXXXX / ETA: HH:ss / Prob: この試行回数でゾロ目が初めて出る確率
```

number of trialsは現在の試行回数を15桁で表示すること．
ETAは経過時間を表示すること．
各試行ごとにテキストを削除して上書きすること．
